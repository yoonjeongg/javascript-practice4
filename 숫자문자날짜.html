문제 1

나쁜 말을 정의한 badWords 배열이 있다.
checkBadWords 함수에 문자열을 매개변수로 넘기면 나쁜 말이 포함 되었는지 여부를 badWords 배열 안의 값 기준으로 확인한 뒤
포함 되어 있으면 "바르고 고운말을 사용합시다" 를 반환하고
포함 되어 있지 않으면 매개변수로 넘겼던 문자열을 그대로 반환하는 checkBadWords 함수를 작성한다.

문제 2

prompt 창으로 게시글 제목을 입력 받고 주어진 해당 게시글 제목이 MAX_LENGTH를 넘을 경우 이후 문자열을 ... 처리하여 반환하는 makeTitle 함수를 작성한다.
(MAX_LENGTH 초과하지 않았을 경우는 문자열 그대로 반환하고 prompt로 문자열 입력이 일어나지 않았을 경우는 "fail" 이라는 문자열을 반환한다.)
EX. prompt 입력 : 오늘 날씨가 참 좋네요 어디 외출 하시나요? , alert 출력 : [오늘 날씨가 참 좋네요 어…] (이)라는 글이 게시되었습니다!
EX. prompt 입력 취소 , alert 출력 : [fail] (이)라는 글이 게시되었습니다!

prompt로 사용자에게 입력 받은 년, 월, 일을 통해 Date 객체를 생성한 뒤
Date 객체를 매개변수로 받아 어떤 요일인지 반환해주는 함수 getWeekDay를 작성하여 호출 테스트한다(결과 콘솔 출력).

문제 4

아래 버튼을 클릭하면 클릭한 순간을 기준으로 오늘 하루 몇 초가 흘렀는지, 오늘 하루 몇 초가 남았는지를 alert로 출력한다.
Ex. 오늘 하루는 몇 초가 흘렀나요? 버튼 클릭, "45912초나 흘렀습니다. 시간을 소중히 씁시다^^" alert 출력
Ex. 오늘 하루는 몇 초가 남았나요?? 버튼 클릭, "40439초 밖에 안 남았습니다. 다시는 오지 않는 오늘입니다^^" alert 출력
